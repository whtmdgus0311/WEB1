-줄바꿈, 단락-


줄바꿈 태그 : <br>
단락 태그 : <p>
<p style="margin-top:숫자px;"> (굵은 글씨는 "속성")

-이미지 넣기-


이미지 태그 : <img> (속성과 함께 사용)
<img src=" ">
<img src="kakao.jpg" width="10%"> (퍼센트는 윈도우 크기 기준)

-목차 만들기-


태그로 목차 만들기 : <li>

<ul> (점 태그)
<li>1.Html</li>
<li>2.CSS</li>
</ul>

  • 1.Html
  • 2.CSS
<ol> (숫자 태그)
<li>Html</li>
<li>CSS</li>
</ol>

  1. Html
  2. CSS

-HTML 구조-


웸 페이지 제목 : <title>
UTF-8 방식으로 저장된 상태 : <meta charset="utf-8">
본문을 설명하는 헤드 태그 : <head> (title,meta charset 등등)
본문 바디 태그 : <body>
최상위 태그 : <html>
이 문서에는 HTML이 담겨 있다는 뜻 : <!DOCTYPE HTML>

-링크 만들기-


링크 다는 태그 : <a> a에 속성 추가 : <a href="주소" target="_blank"(새 탭에서 열기) title="링크 커서 툴팁">
1.html로 가기 : <a href="1.html"></a>
